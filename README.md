# Mobile-First Data Science Portfolio 📱🤖

Welcome to my data science portfolio! This repository contains a collection of real-world analytical pipelines, automation engines, and machine learning models built entirely from scratch using a smartphone screen via Google Colab. 

This sprint proves that high-level data skills don't require expensive hardware—just a browser, curiosity, and execution.

---

## 🛒 Project 1: Retail Inventory Automation Pipeline
**Role:** Supply Chain Data Engineer  
**Objective:** Automate inventory tracking for an e-commerce storefront to flag critical product stock shortages instantly.

### ⚙️ Core Logic & Architecture
* **Tools Used:** Python, Pandas
* **Data Engineering:** Constructed automated data pipelines to track reorder thresholds across dynamic inventory lists.
* **Outcome:** Created a conditional monitoring script that auto-flags supply deficiencies, ensuring the business never runs out of high-demand stock.

## 👥 Project 2: HR Employee Attrition Engine
**Role:** People Analytics Consultant  
**Objective:** Analyze workforce retention data to pinpoint the exact departments and experience levels suffering from high turnover.

### ⚙️ Core Logic & Architecture
* **Tools Used:** Pandas, Advanced Data Aggregation
* **Analytics Logic:** Grouped and cross-referenced multi-variable employee databases to calculate segment-specific resignation rates.
* **Outcome:** Isolated high-risk turnover clusters, providing HR leadership with data-driven insights to fix retention bottlenecks.

## 📈 Project 3: Algorithmic Momentum Trading Bot
**Role:** Quantitative Financial Analyst  
**Objective:** Program an automated trading engine that generates buy and sell stock signals based on moving price trends.

### ⚙️ Core Logic & Architecture
* **Tools Used:** Pandas, Vectorized Math
* **Financial Logic:** Engineered rolling average indicators to track momentum shifts in historical market price data.
* **Outcome:** Built a backtesting logic system that automatically triggers trading signals whenever a stock's short-term price trend crosses its long-term average.

## 🏠 Project 4: Real Estate ML Predictor
**Role:** PropTech Data Scientist  
**Objective:** Build a predictive machine learning model to estimate housing values based on neighborhood demographic patterns.

### ⚙️ Core Logic & Architecture
* **Tools Used:** Scikit-Learn, Linear Regression, Matplotlib
* **Algorithm:** Trained a **Linear Regression** model to map correlations between neighborhood median income levels, home ages, and market values.
* **Outcome:** Successfully created a live inference function capable of generating instant, automated real estate appraisals for new properties.

## 🏥 Project 5: Healthcare ER Operations Optimizer
**Role:** Hospital Operations Analyst  
**Objective:** Analyze emergency room patient arrival cycles to resolve critical wait-time bottlenecks and optimize physician shifts.

### ⚙️ Core Logic & Architecture
* **Tools Used:** NumPy, Capacity-Planning Logic
* **Bottleneck Analysis:** Isolated operational crisis points where patient wait times spiked to 125 minutes (exceeding the 60-minute safety threshold).
* **Capacity Engine:** Programmed resource allocation logic utilizing `numpy.ceil` to dynamically calculate the precise number of doctors required on shift per hour to stabilize wait times.
